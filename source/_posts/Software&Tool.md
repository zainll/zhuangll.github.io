## GitHub Pages + Hexo使用及配置

github上创建一个  username.github.io 的工程，username 必须为github的用户名

<!--more-->

### Hexo 安装

```sh
# 清理缓存
hexo clean
# 生成网页
hexo g
# 启动本地服务端口
hexo s
# 发布到github
hexo d


# 强制推送备份源码分支
git push -f origin backup
```
